# 更新日志

本文件记录 **每日饮食与花销**（`daily-diet-app`）面向用户的变更。技术细节见仓库内代码与提交历史。

## 2025-03-26

- **设置**：移除与每日饮食无关的配置（地点搜索、活动类型、携带物品分类等）。
- **个人体质**：可在设置中填写体重、身高、年龄、性别、活动量与备注；分析时会传入 LLM，用于定性判断当日热量是否相对合适（提示词中含非医疗免责声明）。
- **常用饮食**：在设置中维护列表；新建/编辑每日饮食时，每条餐食可选常用项并「填入本条」。
- **配置日志**：LLM 配置、评分阈值、体质、常用饮食在保存后约 1.2 秒防抖写入本地变更日志；导出、覆盖/合并导入也会记一条。支持在设置页查看与清空日志。
- **数据导出**：备份 JSON 版本升为 **v2**，包含 `diet_user_profile`、`diet_favorite_meals`、`diet_settings_log`；不再导出 geocoding / 行程类应用配置。导入仍兼容旧版备份中的多余字段。
